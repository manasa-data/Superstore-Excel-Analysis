# Superstore Sales Analysis — Excel

## 📊 Project Overview
Sales performance analysis of Superstore dataset using Microsoft Excel.

## 🛠️ Tools Used
- Microsoft Excel

## 📁 Dataset
- Source: Kaggle — Superstore Sales Dataset
- Rows: 500(sample)

## 💡 Key Insights
1. West region generates highest revenue
2. Technology category drives most sales
3. November and December are peak months
4. Consumer segment spends the most
5. Standard Class is most preferred shipping mode

## 📌 Features
- Data Cleaning
- Pivot Tables
- Interactive Dashboard
- Slicers for Region, Category and Year
- KPI boxes — Total Sales, Orders, Avg Order Value
